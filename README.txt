**DEV INSTRUCTIONS**
(1) Get your API key from https://cloud.walletconnect.com/, its free. You can sign in with your web3 wallet or with an email. Then create an test app to get your key.
(2) Copy the .env.local.example file to .env.local
(3) Copy the key from (1) and set it to VITE_WALLETCONNECT_CLOUD_ID={WALLETCONNECT_KEY}
(4) Run `yarn` and `yarn dev`, the dapp should now be running on localhost:5173git
